# PawNest Adoption Platform

## Project Name
**PawNest** – Pet Adoption Platform

## Purpose
PawNest is a full-stack web platform designed to connect loving families with pets available for adoption. Users can browse pets, view detailed profiles, submit adoption requests, and manage their requests. Pet owners or shelters can manage listings and handle adoption requests seamlessly.

## Live URL
[https://paw-nest-client.vercel.app/](https://paw-nest-client.vercel.app/)

## Features
- Browse all available pets with search, filter, and sorting functionality.
- View detailed pet profiles and submit adoption requests.
- Authenticated users can manage their own adoption requests.
- Pet owners can add, update, and delete pet listings.
- Adoption requests are managed with approval/rejection workflow.
- Responsive dark-themed design suitable for desktop, tablet, and mobile.
- Instant search and filter with live updates of pet listings.

## NPM Packages Used

**Frontend (Client)**
- `react` – UI library
- `react-router-dom` – Routing
- `react-hot-toast` – Notifications
- `lucide-react` – Icons
- `motion` – Animations
- `tailwindcss` – Styling
- `typescript` – Type checking

**Backend (Server)**
- `mongodb` – Database
- `cookie-parser` – Cookies support
- `jsonwebtoken` – JWT authentication
- `dotenv` – Environment variables
